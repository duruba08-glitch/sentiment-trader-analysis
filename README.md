# Trader Performance vs Market Sentiment Analysis  

## Overview  
This project analyzes the relationship between Bitcoin market sentiment (Fear/Greed) and trader behavior on Hyperliquid. The goal is to identify patterns in trading performance and generate actionable insights for better trading strategies.

---

## Objective  
- Understand how market sentiment affects trader performance  
- Analyze behavioral changes during Fear vs Greed periods  
- Generate data-driven strategy recommendations  

---

## Datasets  

### 1. Bitcoin Market Sentiment  
- Columns: Date, Classification (Fear/Greed)  

### 2. Hyperliquid Trader Data  
- Fields include: account, symbol, execution_price, size, side, time, closedPnL, leverage, etc.
